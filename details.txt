🏢 PROJECT OVERVIEW
===================
Chetta's Dosa is a full-stack restaurant management system designed for dine-in 
restaurants. It provides a complete digital solution for customers to browse menu, 
place orders, track their orders in real-time, and for restaurant staff to manage 
orders, inventory, employees, analytics, and overall operations.

Primary Color: #C87316 (Orange/Brown - warm and inviting)
Background: #FFFAEF (Off-white, easy on the eyes)
Design: Mobile-first responsive design with bottom navigation

🎯 KEY WORKFLOWS
=================

CUSTOMER ORDER WORKFLOW:
1. Customer scans QR code → Lands on menu
2. Browse categories and items
3. Add items to cart
4. Select table number
5. Apply coupon/loyalty points
6. Place order
7. Order sent to admin + Socket notification
8. Real-time status updates:
   - Pending → Confirmed → Preparing → Ready → Served
9. Request bill
10. Admin generates bill
11. Customer pays (cash/online)
12. Admin marks as paid
13. Loyalty points credited
14. Order completed

ADMIN ORDER MANAGEMENT:
1. Receive new order notification (sound + Socket)
2. View order details
3. Confirm order → Customer notified
4. Mark preparing → Customer notified
5. Mark ready → Customer notified
6. Mark served → Customer notified
7. Customer requests bill
8. Generate bill with all details
9. Customer pays
10. Mark as paid
11. Table becomes available
12. Analytics updated

INVENTORY MANAGEMENT WORKFLOW:
1. Add inventory items
2. Set minimum stock levels
3. Track current stock
4. Receive low stock alerts
5. Restock items
6. Update payment to supplier
7. Generate stock reports

EMPLOYEE MANAGEMENT WORKFLOW:
1. Add employee details
2. Set role and salary
3. Mark daily attendance
4. Track check-in/out times
5. Add holidays
6. Generate monthly reports
7. Calculate salary based on attendance

================================================================================
                         ENVIRONMENT CONFIGURATION
================================================================================

REQUIRED ENVIRONMENT VARIABLES:

Backend (.env):
- NODE_ENV=production
- PORT=5000
- MONGODB_URI=mongodb://localhost:27017/chettas-dosa
- JWT_SECRET=your-secret-key
- FRONTEND_URL=https://yourdomain.com

SMS/OTP:
- SMS_PROVIDER=twofactor
- SMS_API_KEY=your-api-key
- OTP_LENGTH=6
- OTP_TTL_SECONDS=300
- OTP_RESEND_SECONDS=30

Cloudflare R2:
- R2_ACCOUNT_ID=your-account-id
- R2_ACCESS_KEY_ID=your-access-key
- R2_SECRET_ACCESS_KEY=your-secret-key
- R2_BUCKET_NAME=your-bucket
- R2_PUBLIC_URL=https://your-bucket-url

Redis (Optional):
- REDIS_URL=redis://localhost:6379
